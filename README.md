### Install

Check Ruby version: 

	ruby -v

If it's not 2.5.1 then run:

	rbenv install 2.5.1

Install bundler:

	gem install bundler 

Install all packages using bundle:

	gem blundle install

Create and initialize database:

	rails db:create 
    rails db:seed 
    rails db:migrate
 
Run the app locally:

	rails server 
    
Run the test:

	rails test
